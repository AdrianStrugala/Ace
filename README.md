# Ace

Ace is Artificial Intelligence agent enhancing possibilities of every personal computer. Contains
modules responsible for live chat, web browsing and managing programs.

## Technology: 
  * Python
  * Tensorflow
  * SQLite

## Requirements:
  * python 3.6
  * tensorflow-gpu 1.4
  * NVIDIA CUDA 8.0
  * NVIDIA cuDNN 6.0
